Dashboard de Clientes | Análise Fintech

> Projeto de análise de dados focado em comportamento financeiro e geração de insights estratégicos.

---

## Visão Geral

Este projeto simula um dashboard utilizado por fintechs para entender como os clientes utilizam seus recursos financeiros.

A análise foi construída com foco em **tomada de decisão baseada em dados**, explorando padrões de consumo e segmentação de clientes.

---

##  Objetivo

- Identificar clientes de alto valor  
- Entender distribuição de gastos  
- Analisar evolução financeira ao longo do tempo  
- Criar segmentação de clientes  

---

##  Tecnologias Utilizadas

- 🐍 Python  
- 📊 Pandas  
- 📈 Plotly  
- ☁️ Google Colab  

---

##  Principais Insights

- Uma pequena parcela dos clientes concentra a maior parte dos gastos  
- Existem categorias dominantes no comportamento de consumo  
- A segmentação permite personalização de estratégias financeiras
